## Topics Covered

### 1. Database Management
- **Creating a database** with specific file paths for data and log files.
- **Backing up a database** to a `.bak` file.
- **Dropping a database** and understanding how it affects the backup.

### 2. Table Management
- **Creating tables** with primary keys and foreign keys.
- **Altering tables**: adding, modifying, and dropping columns.
- **Data types** used: `INT`, `BIGINT`, `VARCHAR`, `DATETIME`.

### 3. Data Manipulation (DML)
- **INSERT**: adding single and multiple rows.
- **UPDATE**: updating specific rows, all rows, incrementing values, updating multiple columns.
- **DELETE**: deleting specific rows or all rows.

### 4. Data Querying (DQL)
- **SELECT**: retrieving all or specific columns.
- **WHERE clauses**: filtering using `AND`, `OR`, `IN`, `BETWEEN`, `NULL`.
- **ORDER BY**: sorting data ascending/descending, multiple columns.
- **DISTINCT**: avoiding duplicate rows.
- **Concatenation**: combining text and numeric values with `CONVERT` and `SPACE()` functions.
- **Aliases**: using `AS` or brackets/quotes for column names with spaces.
